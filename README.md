SUBDOMAIN_ENUMERATE

аргументы:

    domain – домен сайта у которого ищем поддомены

    -crt – поиск поддоменов с помощью сервиса crt.sh

    -json – вывод в json формате

    -output name – сохранение найденных поддоменов в файл с именем name

изначально программа будет искать с помощью brute-forse то-есть просто перебором (из ~100 популярных поддоменов). очень быстрый способ, но не даст все поддомены. с помощью -crt поиск происходит благодаря сертификатам SSH/TLS. это занимает значительно больше времени, но результат сильно больше чем обычный перебор.

запуск осуществляется:
    python -m app.main test.ru
    где после домена пишутся нужные аргументы
